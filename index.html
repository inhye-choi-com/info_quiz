단원평가 퀴즈 (정보 2)

1. 데이터 수집에 관한 설명으로 알맞은 것을 보기에서 있는 대로 고르시오.
[그림 1 삽입란]
[보기]
㉠ 관찰, 인터뷰, 설문 조사, 센서를 이용하는 방법은 데이터를 직접 수집하는 방법이다.
㉡ 검색 엔진과 데이터 전문 웹 사이트를 이용하면 목적에 맞는 데이터를 수집할 수 있지만 시간과 비용이 많이 든다.
㉢ 데이터를 수집할 때는 데이터의 객관성, 데이터의 품질, 데이터의 정확성을 확인해야 한다.
해설: ㉠과 ㉢은 올바른 설명입니다. ㉡ 검색 엔진 등을 이용한 간접 수집 방법은 직접 수집하는 것에 비해 오히려 시간과 비용을 절약할 수 있습니다.
2. 데이터의 특성에 따른 종류와 활용에 관해 바르게 설명한 친구의 이름을 쓰시오.
[그림 2 삽입란]
- 우현: 수치형 데이터의 예에는 성별, 혈액형, 성취 등급이 있어.
- 예성: 키, 몸무게, 전자 우편 수는 범주형 데이터라고 할 수 있어.
- 은지: 수집한 데이터를 체계적으로 구분하면 데이터의 활용도를 높일 수 있어.
해설: 우현이가 말한 성별, 혈액형 등은 '범주형 데이터'이며, 예성이가 말한 키, 몸무게 등은 '수치형 데이터'입니다. 은지의 설명만 올바릅니다.
3. 다음 빈칸에 들어갈 알맞은 말을 쓰시오.
동영상은 여러 개의 프레임을 빠른 속도로 넘겼을 때 생기는 (        )(을)를 이용한 것이다.
해설: 동영상은 정지된 여러 장의 사진(프레임)을 빠르게 연속으로 보여주어 눈의 잔상 효과를 이용해 움직이는 것처럼 보이게 하는 원리입니다.
4. 다음 빈칸에 들어갈 알맞은 말을 쓰시오.
(        )은/는 합리적 의사 결정을 위해 수집한 데이터의 의미를 해석하여 유용한 정보를 찾아내는 과정을 의미한다.
해설: 수집된 데이터를 목적에 맞게 구조화하고 패턴을 찾아 의미를 도출하는 과정을 데이터 분석이라고 합니다.
5. 계층형 다이어그램 구조화 방법에 대한 설명이나 예시로 가장 적절한 것은?
[그림 3 삽입란]
해설: 계층형 다이어그램은 상하 관계나 분류 체계가 명확한 조직도, 폴더 구조 등을 나타낼 때 가장 적합합니다. 2번은 그래프형 다이어그램, 3번은 표(또는 데이터베이스)에 대한 설명입니다.
